# 기존 스크립트 수정

기존 Windows 11 최적화 스크립트를 수정합니다.

## 사용법
```
/update-script {번호} {수정 내용}
```

## 작업 순서

1. **스크립트 읽기**: ps_scripts/{번호}.*.ps1 파일 읽기
2. **스크립트 수정**: 요청된 기능 추가/수정
   - 기존 스크립트 패턴 유지
   - 단계 번호 [N/M] 업데이트
   - 완료 메시지 섹션 업데이트
3. **README.md 업데이트**: 해당 스크립트 섹션 수정 (필요시)
4. **커밋 및 푸시**: 자동으로 git add, commit, push 수행

## 입력
$ARGUMENTS
